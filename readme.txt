Настроить пользователя Git.

Создать внутри папки файл readme.txt через терминал.

Фиксирование изменений файлов
Посмотрим, как будут меняться состояния изменяемых файлов. Откройте один из добавленных в репозиторий файлов и измените его. Например, добавьте в него ещё одну строку. Выполните команду git status. На экране появится информация о том, что:

файл изменен;
изменения не подготовлены для коммита.
